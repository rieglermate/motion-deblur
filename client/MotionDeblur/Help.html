Az alkalmazás használata

Az alkalmazás használatához először ki kell választania egy képet a "Kép kiválasztása..." gomb segítségével. Ezt követően nyílik lehetősége lekérés indítására.

Az "Indítás" gombbal indíthat egy lekérdezést a képfeldolgozó API (Application Programming Interface) felé. Amennyiben sikeresen megérkezett a válasz, két kép fog megjelenni az ablak közepén:

  • bal oldalon a 256x144 felbontásra átméretezett eredeti kép
  • jobb oldalon pedig a modell eredménye az átméretezett eredeti kép bemenetre
Ezután a "Kép mentése..." gombokkal mentheti el egyenként a megjelenített képeket. Sikeres mentés után a képfájlokat szokásos módon megnyithatja bármely erre alkalmas programmal.

Minden művelet végrehajtása során visszajelzéseket fog kapni, ezeket az ablak státuszsorán olvashatja el.
